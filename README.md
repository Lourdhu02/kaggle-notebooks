## How to Use

Each notebook file name is the SEO-optimized Kaggle title. When publishing:

1. Open the `.ipynb` file
2. Use the filename (without `.ipynb`) as the Kaggle notebook title
3. Add matching tags on Kaggle (topic + difficulty)
4. Run `python update_progress.py` after publishing
5. Push to git
